# ManusAI
- **URL:** https://manus.im/app
- **Purpose:** Advanced agent for creating slides, websites, apps, and design.
- **Integration:** Accessed via browser tab.
- **Capabilities:**
    - **Image Creation:** Generate custom visuals from text (perfect for Instagram posts/stories).
    - **Web Browsing:** Research competitors, hashtags, and trends.
    - **Content:** Write captions, copy, and brand messaging.
    - **Previous projects:** "AurenAI Content Engine".

### Workflows
- **Instagram Visuals:** Ask ManusAI to "Create a 5-slide carousel about [topic] for Instagram".
- **Captions:** Ask ManusAI to "Write a caption for this image about [topic]".
